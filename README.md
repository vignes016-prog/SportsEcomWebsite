<h1>🏆 Champions Corner - Sports E-commerce Website </h1>

A responsive front-end demo website for a sports equipment store, built with HTML, CSS, and Bootstrap 5. This project showcases a complete user flow from login to product browsing.

🔗 Live Demo: https://vignes016-prog.github.io/SportsEcomWebsite/

📋 Features


Login Page — Simple client-side authentication flow with form validation
Home Page — Hero section with call-to-action
Product Catalog — Responsive product cards for sports categories (Cricket, Football, Basketball, Volleyball, Kabaddi)
About Us Section — Company details and story
Contact Section — Contact info with social media links
Customer Feedback — Image carousel with a feedback form
Fully Responsive — Optimized layout for mobile, tablet, and desktop screens


🛠️ Built With


HTML5
CSS3 (Custom styling with media queries for responsiveness)
Bootstrap 5.3.8
Fontawesome for icons
javascript for validation
